# Architecture

## Objective

Build an enterprise People Analytics Lakehouse using Azure Databricks, Unity Catalog and Delta Lake.

## Technology Stack

- Azure Databricks
- Unity Catalog
- Azure Data Lake Storage Gen2
- Delta Lake
- PySpark
- GitHub
- Power BI (next stage)

## Architecture

Raw CSV files
      ↓
Azure Data Lake Storage
      ↓
Bronze Layer
      ↓
Silver Layer
      ↓
Gold Layer
      ↓
Power BI
